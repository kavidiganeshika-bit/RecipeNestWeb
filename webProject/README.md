# RecipeNest Web Application

## Project Description

RecipeNest is a simple web-based recipe management system developed using HTML, CSS, Bootstrap, PHP, and MySQL. It allows users to register, log in, and send messages through a contact form.

## Features

* User Registration and Login System
* Session Management (Login/Logout)
* Contact Form with Database Storage
* Responsive Design using Bootstrap

## Setup Instructions

### 1. Install XAMPP

Download and install XAMPP and start Apache and MySQL.

### 2. Place Project Folder

Copy the project folder into:
C:\xampp\htdocs\

### 3. Import Database

* Open browser and go to: http://localhost/phpmyadmin
* Create a database named: recipe_db
* Click Import
* Select the file: database.sql
* Click Go

### 4. Run the Project

Open browser and go to:
http://localhost/webProject
